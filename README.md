# شكيب نت — نظام إدارة وكيل الإنترنت

نظام ويب لإدارة اشتراكات وحسابات وكيل الإنترنت (Next.js).

- **الويب/API**: Vercel → https://shakeebnet.com
- **قاعدة البيانات**: Neon (PostgreSQL)
- **العامل المحلي**: واتساب + المزامنة (يعمل بـ RUN_WORKER=1 على حواسيب المكاتب)

النشر تلقائي عند الدفع إلى فرع `main`.
