## ثالثاً: الرفع على الانترنت (استضافة + دومين)

### الخيار (أ) — الأسهل: Vercel + قاعدة سحابية
1. أنشئ قاعدة PostgreSQL مجانية على [Neon](https://neon.tech) أو [Supabase](https://supabase.com) واحصل على رابط الاتصال.
2. ارفع المشروع إلى GitHub.
3. اربطه بـ [Vercel](https://vercel.com) واضبط متغيّرات البيئة (أدناه).
4. طبّق المخطط على القاعدة السحابية:
   ```bash
   npx prisma migrate deploy
   ```
5. اربط الدومين من إعدادات Vercel (SSL تلقائي).

### الخيار (ب) — خادم VPS خاص
1. استأجر VPS (Ubuntu) وثبّت Node.js 20+ و PostgreSQL.
2. انسخ المشروع، اضبط `.env`، ثم:
   ```bash
   npm install
   npx prisma migrate deploy
   npm run build
   npm start        # أو عبر pm2 للتشغيل الدائم
   ```
3. ضع Nginx كوكيل عكسي + شهادة SSL عبر Certbot.
4. وجّه الدومين إلى عنوان الخادم.

## متغيّرات البيئة (`.env`) للإنتاج

```env
# رابط قاعدة البيانات السحابية
DATABASE_URL="postgresql://user:pass@host:5432/dbname?sslmode=require"

# مفتاح تشفير الجلسات — ولّد مفتاحاً عشوائياً قوياً
AUTH_SECRET="<مفتاح-عشوائي-طويل>"

# مزوّد الرسائل (mock = محاكاة، غيّره عند ربط مزوّد حقيقي)
MESSAGING_PROVIDER="mock"
```

> ⚠️ **مهم للإنتاج:** غيّر كلمة سر المستخدم `admin`، واستبدل `AUTH_SECRET` بمفتاح عشوائي قوي.
